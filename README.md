# SillyTyper

Just types the contents of an input file with a time delay after each line.

## Getting Started

inpuutfile.txt:
one,1000
two,2000

Output on console:
$python sillytyper.py inputfile.txt
one [1000 miliseconds delay]
two [2000 miliseconds delay]
